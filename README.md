# reminder-app

Small open-source reminder application written in Go.

Built with:
- Go
- Fyne (GUI) — https://fyne.io/

## Features
- Simple reminder management
- Desktop GUI

> If you’re adding features, please update this list.

## Requirements
- Go (version from `go.mod`)
- Fyne prerequisites for your OS (see Fyne docs)

## Run (dev)
```bash
go run .
```

# Install
## Linux
* ```sudo apt-get update```
* ```sudo apt-get install build-essential```

Set up the environment variables ```CGO_ENABLED=1```
